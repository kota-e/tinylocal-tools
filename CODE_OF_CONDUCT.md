# Code of Conduct

## Our Standard

Be kind, clear, and patient.

Helpful behavior includes:

- Respecting different skill levels.
- Giving useful feedback.
- Staying focused on the work.
- Assuming good intent.

Unacceptable behavior includes:

- Harassment or insults.
- Threats.
- Discriminatory language.
- Sharing private information without permission.

## Enforcement

Maintainers may remove content, reject contributions, or block users who break
this code of conduct.

Reports can be sent to the maintainers by email or through GitHub moderation
tools.
