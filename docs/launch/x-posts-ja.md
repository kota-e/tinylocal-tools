# TinyLocal Tools X投稿案（日本語）

## 短文案

### 案1: 文章を貼る前の小さな安心

文章をAI、チャット、GitHub Issue、メールに貼る前に、
改行を直したり、見せたくない情報をかくしたり、Markdown表を整えたりできる小さな道具を作りました。

TinyLocal Tools
- PasteFix
- SafePaste
- Markdown Table Fixer

ブラウザ内で動き、文章はアップロードしません。

### 案2: コピペ前の困りごと

PDFやWebからコピーした文章、変なところで改行されがち。
共有前の文章、メールアドレスやキーが混ざっていないか不安。
Markdown表、少し崩れるだけで読みにくい。

その3つを、ブラウザ内で直す小さなアプリを作りました。
TinyLocal Toolsです。

### 案3: プライバシー重視

TinyLocal Toolsを作りました。

文章をきれいにする、共有前に危ない情報をかくす、Markdown表を直す。
その処理をサーバーに送らず、ブラウザの中で行うための小さなツール集です。

アカウントなし、解析なし、外部AI APIなし。

### 案4: 開発者・書く人向け

ドキュメント、問い合わせ文、バグ報告、AIへの貼り付け。
「貼る前にちょっと整えたい」場面が多いので、TinyLocal Toolsを作りました。

PasteFixで文章を整え、
SafePasteでよくある秘密っぽい文字をかくし、
Markdown Table Fixerで表を直せます。

### 案5: SafePaste注意つき

SafePasteは、メールアドレス、電話番号、APIキーっぽい文字などを見つけてかくす道具です。

ただし、完璧な匿名化ではありません。
「見落としを減らすための手伝い」として使い、共有前に必ず自分の目で確認してください。

TinyLocal Toolsの中で使えます。

### 案6: OSS向け

TinyLocal Toolsは、文章を扱うための小さなOSSブラウザアプリです。

- 汚れたコピー文を整える
- 共有前に危ない情報をかくす
- Markdown/CSV/TSVの表を直す

バックエンド、DB、解析、テレメトリ、外部AI APIはありません。
文章はブラウザ内で処理します。

## スレッド案

### 1/5

TinyLocal Toolsという小さなブラウザアプリを作りました。

目的はシンプルです。
「文章をどこかに貼る前に、きれいにして、危ない情報を減らして、共有しやすくする」ことです。

### 2/5

PasteFixは、PDF、Webページ、メール、メモからコピーした文章を整える道具です。

変な改行、余分な空白、読みにくい段落を直して、メール、チャット、AIツール、ドキュメントに貼りやすくします。

### 3/5

SafePasteは、共有前の文章に混ざった危ない情報をかくす道具です。

メールアドレス、電話番号、APIキーっぽい文字など、よくあるパターンを見つけます。

ただし、これは完璧な匿名化ではありません。
最後は必ず自分の目で確認してください。

### 4/5

Markdown Table Fixerは、Markdown、CSV、TSVの表を直す道具です。

少し崩れた表を整えて、README、Issue、Zenn、Qiita、Obsidianなどに貼りやすくします。

### 5/5

TinyLocal Toolsはローカルファーストです。

バックエンドなし。
DBなし。
解析なし。
テレメトリなし。
アカウントなし。
外部AI APIなし。

文章は基本的にブラウザ内で処理し、アプリ側からサーバーへアップロードしません。
