# サーバーに文章を送らずに、コピペ前の整形とマスクをするTinyLocal Toolsを作りました

## これは何か

TinyLocal Toolsは、文章を「貼る前」に整えるための小さなブラウザアプリです。

主な道具は3つです。

- PasteFix: PDFやWebからコピーした文章の改行・空白を整える
- SafePaste: 共有前の文章にある、秘密っぽい文字をマスクする
- Markdown Table Fixer: Markdown、CSV、TSVの表を整える

技術的には、Vite、React、TypeScript、CSSで作った静的アプリです。
バックエンド、DB、解析、テレメトリ、アカウント機能、スクレイピング、外部AI APIはありません。

## 解きたい問題

文章を扱う作業では、地味だけれどよく起きる問題があります。

### コピーした文章が汚れる

PDFやWebページからコピーすると、次のようなことがあります。

- 変な位置で改行される
- 余分な空白が入る
- 段落が読みにくくなる

このままAIツール、メール、チャット、ドキュメントへ貼ると、読む側も直す側も少しつらくなります。

### 共有前に秘密が混ざる

問い合わせ、ログ、バグ報告、社内メモを共有するとき、文章の中に次のような情報が混ざることがあります。

- メールアドレス
- 電話番号
- URL
- APIキーのように見える文字
- その他、見せたくない固有の言葉

人の目だけで毎回確認するのは大変です。

### Markdownの表が崩れる

README、Issue、Qiita、Zenn、Obsidianなどでは、Markdown表をよく使います。

でも、CSVやTSVから持ってきた表、手で直した表は、少し崩れるだけで読みづらくなります。

## TinyLocal Toolsの方針

TinyLocal Toolsでは、機能を増やしすぎるよりも、次の方針を大事にしています。

- 小さく保つ
- 読みやすく保つ
- 文章処理のロジックをUIから分ける
- ブラウザ内で処理する
- 入力履歴を保存しない
- ユーザーの文章を勝手に送らない

「大きなサービス」ではなく、「手元で使う小さな道具」を目指しています。

## プライバシー面

TinyLocal Toolsの大事な価値は、ローカルファーストであることです。

アプリのコードはブラウザで動きます。
貼り付けた文章を、TinyLocal Tools側のサーバーへ送る作りにはしていません。

入れていないものは次の通りです。

- バックエンド
- データベース
- アクセス解析
- テレメトリ
- アカウント機能
- スクレイピング
- 外部AI API

静的サイトとして公開した場合、ホスティング先の通常のアクセスログは別の話として存在し得ます。
ただし、TinyLocal Tools自体は入力文を集めたり、保存したりする目的の処理を持ちません。

## 各ツール

### PasteFix

PasteFixは、コピーした文章を整えるためのツールです。

たとえば、PDFからコピーした文章が行ごとに切れているとき、貼り付け先で読みやすい形に直す手伝いをします。

使う場面の例です。

- AIツールに貼る前
- メールやチャットに貼る前
- GitHub Issueを書く前
- ドキュメントに引用を入れる前

### SafePaste

SafePasteは、文章の中にある秘密っぽい情報をマスクするためのツールです。

メールアドレス、電話番号、APIキーのように見える文字など、よくあるパターンを見つける手伝いをします。

重要な注意点があります。

SafePasteはベストエフォートです。
つまり、できるだけ見つける道具ですが、完璧な匿名化ではありません。
見落としがあり得ます。
共有する前に、必ず自分の目で確認してください。

### Markdown Table Fixer

Markdown Table Fixerは、Markdown、CSV、TSVの表を直すためのツールです。

次のような場面で使います。

- READMEに表を貼る
- QiitaやZennの記事に表を入れる
- GitHub Issueに調査結果を貼る
- Obsidianのメモに表を残す

表の形を整えることで、読む人が内容に集中しやすくなります。

## 技術メモ

構成はシンプルです。

- Vite
- React
- TypeScript
- plain CSS
- Vitest

文章処理は、Reactコンポーネントの中へ閉じ込めず、純粋関数として分けています。

この形にしておくと、次の利点があります。

- UIを開かなくてもテストしやすい
- 入力と出力を確認しやすい
- 後から動きを変えるときに壊れにくい
- 「見た目」と「文章を直す処理」を分けて考えられる

## 使ってほしい人

TinyLocal Toolsは、次のような人に合うと思います。

- 文章をAIツールへ貼る前に整えたい人
- サポート文やバグ報告を共有する人
- READMEや記事でMarkdown表をよく使う人
- アカウント登録なしで使える小さな道具が好きな人
- 文章を勝手にサーバーへ送らないツールを使いたい人

## まとめ

TinyLocal Toolsは、すごく大きなことをするアプリではありません。

でも、コピー、整形、マスク、表の修正は、毎日の作業で何度も出てきます。
そこを少し楽にして、しかも文章を勝手に外へ送らない。

そのための小さなローカルファーストな道具として作っています。
