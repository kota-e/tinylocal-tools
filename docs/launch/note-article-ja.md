# 貼る前の文章を、少し安心して整える。TinyLocal Toolsを作りました

## はじめに

文章をどこかへ貼る前に、少しだけ不安になることがあります。

PDFからコピーした文章の改行がぐちゃぐちゃ。
Webページから持ってきた文章に、変な空白が入っている。
問い合わせ文やバグ報告を共有したいけれど、メールアドレスや電話番号が混ざっていないか心配。
Markdownの表が崩れて、読む人にやさしくない。

こういう小さな困りごとを減らすために、TinyLocal Toolsを作りました。

## TinyLocal Toolsとは

TinyLocal Toolsは、ブラウザで動く小さな文章ツール集です。

できることは大きく3つです。

- PasteFix: コピーした文章の改行や空白を整える
- SafePaste: 共有前に、危ない情報っぽい文字をかくす
- Markdown Table Fixer: Markdown、CSV、TSVの表を直す

大きなサービスではありません。
アカウントを作る必要もありません。
文章を整えるための、机の上の小さな道具箱のようなものです。

## なぜ作ったのか

文章を扱う作業では、「本題ではないけれど面倒なこと」がよく起きます。

たとえば、PDFからコピーした文章をAIツールに貼ろうとしたら、1行ごとに変な改行が入ってしまう。

たとえば、サポートへの問い合わせ文をチームに共有したいけれど、お客様のメールアドレスや電話番号が入っているかもしれない。

たとえば、Markdownの表をREADMEやIssueに貼ったら、列がずれて読みにくくなる。

どれも小さなことです。
でも、毎日くり返すと少し疲れます。
TinyLocal Toolsは、その「貼る前のひと手間」を軽くするために作りました。

## プライバシーを大事にしました

TinyLocal Toolsは、ローカルファーストの考え方で作っています。

つまり、文章の処理は基本的にブラウザの中で行います。
アプリ側から、貼り付けた文章やファイルをサーバーへアップロードしません。

また、次のようなものは入れていません。

- バックエンド
- データベース
- アクセス解析
- テレメトリ
- アカウント機能
- スクレイピング
- 外部AI API

もちろん、公開されたWebページとして開く場合、ホスティング先の通常のアクセスログが残ることはあります。
でも、TinyLocal Tools自体は、入力した文章を集めたり、あとで見るために保存したりしません。

## 3つの道具

### PasteFix

PasteFixは、コピーした文章を読みやすく整える道具です。

PDF、Webページ、メール、メモアプリなどから文章をコピーすると、変な改行や空白が入ることがあります。
PasteFixは、そうした文章を別の場所へ貼りやすい形にします。

AIツール、メール、チャット、ドキュメント、GitHub Issueなどへ貼る前に使うと便利です。

### SafePaste

SafePasteは、共有前の文章から、見せたくない情報っぽいものを見つけてかくす道具です。

たとえば、メールアドレス、電話番号、URL、APIキーのように見える文字などを見つける手伝いをします。

ただし、SafePasteは完璧な匿名化ツールではありません。
あくまで「見落としを減らすための手伝い」です。
共有する前には、必ず自分の目で結果を確認してください。

### Markdown Table Fixer

Markdown Table Fixerは、Markdown、CSV、TSVの表を整える道具です。

README、Zenn、Qiita、Obsidian、GitHub Issueなどで、表をきれいに貼りたいときに使えます。

列がずれて読みにくい表を、少し見やすい形に直します。

## こんな人に使ってほしい

TinyLocal Toolsは、文章をよくコピーして貼る人に向いています。

- AIツールに文章を貼る前に整えたい人
- メールやチャットに貼る文章を読みやすくしたい人
- 問い合わせ文やバグ報告を共有する前に、危ない情報を減らしたい人
- README、Issue、Zenn、Qiita、ObsidianでMarkdown表をよく使う人
- アカウント登録やトラッキングなしで、さっと使える道具がほしい人

## おわりに

TinyLocal Toolsは、派手なアプリではありません。

でも、文章を貼る前の「ちょっと困った」を減らせると、毎日の作業が少し楽になります。

大事にしたいのは、文章を勝手に外へ送らないこと。
そして、むずかしい設定なしで、必要なときにすぐ使えることです。

小さな道具ですが、文章を扱う人の手元で役に立てばうれしいです。
