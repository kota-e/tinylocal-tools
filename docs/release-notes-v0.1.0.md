# TinyLocal Tools v0.1.0 Release Notes

TinyLocal Tools v0.1.0 is the first public release of a small, local-first text
toolkit for ordinary browser users.

It gives you three simple tools in one static app:

- **PasteFix** cleans messy copied text from PDFs, websites, emails, and notes.
- **SafePaste** masks common sensitive-looking information before you share text.
- **Markdown Table Fixer** repairs and formats Markdown, CSV, and TSV tables.

## Privacy First

TinyLocal Tools is designed so your text stays in your browser by default.

The app has no backend, database, analytics, telemetry, accounts, scraping, or
external AI API calls. It does not silently upload your text or files.

SafePaste is best-effort. It can help reduce sharing risk, but it is not perfect
anonymization. Please review the masked result before sending it to another
person, website, or AI tool.

## What You Can Do

- Clean text before pasting it into AI tools.
- Clean copied PDF or website text with broken lines and spacing.
- Mask sensitive information before sharing a message, support ticket, bug
  report, or note.
- Fix Markdown, CSV, and TSV tables so they are easier to read and paste.

## Known Limits

- SafePaste is a helpful check, not a guarantee.
- Text cleaning can change spacing or line breaks, so review important text.
- Table fixing works best when the pasted table has clear rows and columns.

## For Developers

This release includes the React app shell, pure text-processing logic, unit
tests, English and Japanese UI copy, and open-source project files.
